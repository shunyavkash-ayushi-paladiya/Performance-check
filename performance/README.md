# PulsePilot

PulsePilot is a premium website performance testing dashboard demo inspired by PageSpeed-style analysis tools.

## Folder Structure

```text
.
├── index.html
├── css/
│   └── styles.css
└── js/
    └── main.js
```

## Features

- Unique brand identity and logo mark
- Modern SaaS-style dashboard layout
- URL testing form with device and connection controls
- Simulated performance analysis and loading state
- Performance, accessibility, best practices, and SEO cards
- Core Web Vitals section
- Suggestions and improvement recommendations
- Tested URL history with localStorage
- Inline JSON response viewer with copy action
- Responsive design for all screen sizes

## How to Run

Open `index.html` in a browser or use your local server URL.
